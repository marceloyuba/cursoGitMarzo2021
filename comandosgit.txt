mkdir    crear una carpeta
git init  crea un repositorio local
git status muestra si hay commits
git config -- sirve para configurar
git commit -"***" genera un commit en un repositorio
git log    muestra informacion de los commits
git add agrega un archivo al branch "add ." agrega todo lo que hay
git log --oneline simplifica la info y te da los primeros 7 aracteres del commit
git remote add origin https://github.com/xxxxxxx/xxxxxxxxx.git para establecer una conexion con el github
git push origin main     para subir un repositorio al github

git brunch muestra los brunches en el proyecto
git brunch nombre genera un brunch
git checkout nombre selecciona ese brunch
